- `hud/hud_panel_timer.svg`: scalable coral timer panel.
- `hud/icon_coin.svg`: income/coin icon.
- `hud/icon_clock.svg`: timer icon.
- `hud/icon_star.svg`: currency/reputation icon.

## Order

- `order/order_bubble_selected.svg`: current customer bubble frame.
- `order/order_bubble_normal.svg`: secondary customer bubble frame.
- `order/order_item_slot.svg`: dynamic order item slot.
- `order/heart_icon.svg`: patience icon.
- `order/patience_track.svg`: patience track.
- `order/patience_fill_green.svg`: healthy patience fill.
- `order/patience_fill_amber.svg`: warning patience fill.
- `order/patience_fill_red.svg`: critical patience fill.

Text, values, product icons, and patience amounts remain dynamic in Cocos.
