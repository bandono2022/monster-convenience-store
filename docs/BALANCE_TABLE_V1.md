## 口径

本表只为前 7 天垂直切片服务。目标是“能跑、能测、能调”，不是长期经济模型。

- 单轮营业：90 秒。
- 同屏顾客：最多 2 位。
- 订单件数：前 7 天只做 1-2 件，不做 3 件。
- 交付：订单气泡 READY 后点击顾客或订单气泡，收银台只做入账反馈。
- 货币单位：金币。
- 收入目标指营业额；成本和净利润用于结算与升级。

## 临时 V1 边界

- 不做通胀、贷款、广告、IAP、离线收益。
- 不做精确需求预测，只用热门品类和库存状态提示。
- 不做 3 件订单，不做三顾客。
- 不做多微波炉、多备餐位、多加工设备。
- 解锁价只服务前 7 天可玩；30 天经济后续另表。
