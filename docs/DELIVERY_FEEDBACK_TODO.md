## 建议执行顺序

1. 先实现单托盘服务锁：托盘绑定一个顾客，不随切换瞬变。
2. 普通商品先入托盘，READY 后点击交付。
3. 再把热饭团接入微波炉 -> 托盘，并共享同一个服务锁。
4. 再补订单气泡状态。
5. 最后补交付飞行动画和顾客表情。
