# M1 垂直切片验收清单

更新时间：2026-06-17

## 目标

M1 的目标是让玩家能完成一轮 90 秒营业，并清楚理解：

顾客订单 -> 取货/加工 -> 放入托盘 -> 点击交付 -> 顾客反馈 -> 收银入账 -> 结算。

## 手动验收脚本

1. 从首页进入营业。
2. 确认顶部时间从 90 秒开始倒计时。
3. 观察当前顾客订单是否比另一个顾客更突出。
4. 点击当前订单需要的普通商品，确认商品进入托盘，订单图标打勾。
5. 切换到另一位顾客，确认托盘内容不会变化。
6. 在托盘未完成时点击托盘，确认提示还缺商品并红闪。
7. 点错商品，确认商品卡震动、订单/托盘红闪、顾客生气。
8. 如果订单需要加热，点击饭团进入微波炉，等待完成后点击微波炉，确认热饭团进入托盘。
9. 托盘 READY 后点击托盘，确认商品飞向顾客、订单盖 `DONE`、收入弹出。
10. 故意等待顾客超时，确认顾客生气离开，托盘/微波炉状态被清理。
11. 等 90 秒结束，确认进入结算页并显示营业额、净利润、完成单数、流失顾客。
