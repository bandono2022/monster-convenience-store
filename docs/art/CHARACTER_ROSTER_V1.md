# 怪兽顾客角色阵容 V1

角色设计表：`docs/art/characters/monster-customer-roster-v1.png`

## 统一规则

- 使用深紫色粗描边、柔和赛璐璐明暗和暖色高光。
- 顾客必须仅靠剪影就能区分，后续角色禁止只做换色。
- 每名顾客至少准备正常、开心、焦急、生气四种状态。
- 正式游戏优先使用程序动效：呼吸、弹跳、抖动、前倾和离场。
- 小配饰用于表达性格，但不能遮挡面部和主要轮廓。

## 初始角色

### 红色普通客

- 定位：教学期与日常订单的基准顾客。
- 行为：普通耐心，购买 1 至 2 件商品。
- 资产：继续使用现有红色怪兽四状态素材。

### 黄色急躁客

- 定位：最早出现的压力型顾客。
- 行为：耐心较短，快速服务奖励更高。
- 资产：继续使用现有黄色怪兽四状态素材。

## 后续角色

### 蓝色大单客

- 当前实现：商店等级 2 解锁，约每四名顾客出现一次。
- 行为：购买 2 至 3 件商品，耐心较长，单次利润较高。
- 玩法作用：考验备货结构和连续操作。
- 资产：四状态透明素材位于 `assets/resources/game-art/characters/bulk-shopper/`。

### 粉色热食客

- 解锁建议：解锁微波炉后。
- 行为：更偏好饭团、布丁等需要加工或热食类商品。
- 玩法作用：提升微波炉升级与热食库存的价值。

### 紫色挑剔客

- 解锁建议：商店等级 3。
- 行为：订单数量少，但等待评价和缺货惩罚更高。
- 玩法作用：迫使玩家关注服务速度与准确度。

### 青绿色潮流客

- 解锁建议：热门品类预测系统稳定后。
- 行为：优先购买当日热门品类，成功服务提供额外金币。
- 玩法作用：让营业前预测和进货决策产生直接收益。

## 推荐出场顺序

1. 红色普通客
2. 黄色急躁客
3. 蓝色大单客
4. 粉色热食客
5. 青绿色潮流客
6. 紫色挑剔客
