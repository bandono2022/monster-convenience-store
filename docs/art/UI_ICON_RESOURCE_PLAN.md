# UI 与 Icon 资源汇总

目标：先把营业主界面按目标图还原到可玩的商业手游质感。UI 资源分三类处理：

- 已有运行时素材：直接继续用。
- 代码绘制 UI：短期先用 Cocos `Graphics` 画，方便调比例和交互。
- 正式 PNG/Icon：比例稳定后再统一生成或绘制，避免反复返工。

## 建议资源生成顺序

1. 先不急着生成整套 UI PNG，优先用代码画 HUD、气泡、商品卡、菜单，因为这些还在频繁调比例。
2. 当营业页比例稳定后，统一生成一张 `ui-icons-v1.png`：
   - coin
   - clock
   - star
   - pause
   - plus
   - heart
   - fire
   - check
   - warning
   - sold-out
3. 再生成一张 `ui-effects-v1.png` 或序列帧：
   - sparkle
   - coin-fly
   - heat-glow
   - error-burst
