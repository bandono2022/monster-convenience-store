# 怪兽便利店 V3.1 营业页视觉规范

基准图：`docs/art/concepts/monster-store-v3.1-gameplay-final.png`

## 视觉方向

- 精致商业休闲手游，加入少量手绘包装与便利店细节。
- 面向 16 至 35 岁，可爱但不低龄。
- 使用深紫褐描边、暖米白表面和深绿色便利店品牌色。
- 珊瑚红只强调紧急与错误；黄色只强调金币、完成与重要奖励。
- 背景降低饱和度，订单、顾客状态和可操作商品保持最高对比度。

## 营业页层级

1. 订单内容与耐心值。
2. 顾客表情和当前服务目标。
3. 微波炉、自动付款和交付状态。
4. 商品需求、库存、加热与售罄状态。
5. 停止接客。

## 功能布局

- 顶部：营业额、倒计时、口碑与暂停。
- 顾客区：同时显示两名轮廓明显不同的怪兽；订单气泡不遮挡表情。
- 设备区：左侧仅保留一个微波炉，右侧为不可点击的自动付款设备。
- 交付区：设备与商品之间保留一条窄交付轨道，用于商品飞行动效。
- 商品区：固定六张商品卡，显示库存数量，并支持需要、加热、售罄与锁定状态。
- 停止接客：使用小型营业牌控件，保持次要视觉权重。

## 首批素材拆分

- 场景：便利店营业背景、柜台前景、交付轨道。
- 角色：普通顾客与急性顾客，每名角色包含等待、开心、着急、生气四种表情。
- 设备：微波炉空闲、加热、完成三种状态；自动付款设备空闲与到账两种状态。
- 商品：六种商品的统一正式插画。
- UI：订单气泡、耐心条、商品状态徽章、库存徽章、顶部 HUD 和停止接客牌。

## 动效原则

- 商品点击后飞向顾客；需加热商品先飞入微波炉，再从交付轨道飞向顾客。
- 顾客完成订单后表情切换、轻微弹跳，金币从自动付款设备飞入顶部营业额。
- 售罄时商品卡变灰并短促震动；顾客生气离场，不持续闪烁。
- 单次反馈控制在 0.15 至 0.8 秒，避免拖慢经营节奏。
