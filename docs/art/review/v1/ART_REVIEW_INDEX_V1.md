## 确认时请重点看

1. 角色方向是否认可：普通红怪、急性黄怪、大单蓝怪是否都能作为顾客长期使用。
2. 商品方向是否认可：六个商品的可爱程度、识别度、商业手游感是否够。
3. 设备方向是否认可：微波炉和店内背景是否继续沿用这个风格。
4. 是否接受下一步优先补“状态图标、轻动效、结算反馈”，而不是先做首页外景。

## 不建议现在接入的内容

- 概念图 `docs/art/concepts/`：只作为风格参考，不作为运行时资源。
- UI 截图 `docs/art/ui/`：只作为当前画面记录，不作为素材。
- 商品旧白底图：只作为历史备份，不再作为运行时展示资源。

## 下一步接入顺序建议

1. 先确认本目录 3 张分类总览图。
2. 已生成透明底商品图，并接入运行时。
3. 已制作并接入正式收银台、交付托盘；旧柜台前景因遮挡过高已停用。
4. 下一步补金币、时间、口碑、加热、售罄、需要、已交付等核心 UI 图标。
5. 再补收款、交付、加热完成、顾客情绪变化等轻动效。
